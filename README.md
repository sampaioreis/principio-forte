# ⚔️ Princípio Forte

Princípio Forte é um projeto de reconstrução do caráter masculino
baseado em princípios cristãos, disciplina pessoal e responsabilidade prática.

Este não é um projeto motivacional.
Não é autoajuda.
Não é conteúdo para consumo passivo.

## Por que existe

Homens não falham de uma vez.
Eles se perdem quando vivem sem princípio, sem estrutura e sem responsabilidade.

O Princípio Forte existe para confrontar essa deterioração silenciosa
e oferecer estrutura para homens que não podem mais viver em contradição
entre o que dizem crer e a vida que levam.

## Para quem é

- Homens entre 28 e 45 anos
- Casados ou em relacionamento sério
- Com responsabilidades reais
- Que reconhecem incoerência entre fé, disciplina e prática
- Que entendem que mudança exige custo

## Para quem NÃO é

- Quem busca motivação barata
- Quem terceiriza culpa
- Quem rejeita disciplina
- Quem quer resultado sem responsabilidade

## Princípios inegociáveis

- Fé cristã aplicada à vida real
- Disciplina mínima diária
- Responsabilidade pessoal
- Liderança no lar
- Coerência entre discurso e prática

## Estrutura do projeto

Este repositório contém o site oficial do Princípio Forte,
publicado via GitHub Pages, com conteúdo direto, enxuto e intencional.

O foco não é estética.
O foco é clareza, estrutura e confronto.

## Aviso

Se este conteúdo incomodar, ele está funcionando.
Se gerar conforto, foi mal interpretado.
